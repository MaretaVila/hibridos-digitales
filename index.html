¿Cómo usar este sitio?

Para analizar obras

  • Recorre las fichas para identificar categorías, autores, contextos de producción y modos de circulación.
  • Usa las taxonomías del archivo para comparar prácticas como música visual, arte generativo, narrativas sensoriales y visualización inmersiva.

Para situar el campo

  • Consulta la sección de recursos para ubicar artistas, instituciones, festivales y herramientas vinculadas al ecosistema fulldome.
  • Revisa la página de referencias para comprender cómo estas prácticas circulan en redes internacionales de exhibición e investigación.

Para producir

  • Explora la sección de producción como guía inicial para pensar concepto, composición hemisférica, sonido y flujo de trabajo.
  • Relaciona estas orientaciones con las obras del archivo para observar distintas estrategias de resolución formal y técnica.

Para trabajar en educación

  • Utiliza el sitio como archivo pedagógico para clases, seminarios, talleres y proyectos de investigación-creación.
  • Propón recorridos temáticos según públicos, problemas de análisis o procesos de producción inmersiva.
